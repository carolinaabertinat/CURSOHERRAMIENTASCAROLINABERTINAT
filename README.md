## 🎯 Objetivo  
Analizar el mercado laboral en inteligencia artificial para identificar patrones en salarios, nivel de experiencia, ubicación geográfica y características de los empleos.

## 🧠 Preguntas de análisis
- ¿Cómo varía el salario según el nivel de experiencia?
- ¿Qué roles presentan mayores salarios promedio?
- ¿Qué países concentran más oportunidades laborales?

## 📂 Dataset  
Se utilizó un dataset sintético sobre el mercado laboral en inteligencia artificial (2020–2026), que incluye información sobre roles, salarios, experiencia, educación, habilidades y ubicación de las empresas.

## 🧪 Metodología  
- Carga y limpieza de datos con Python (pandas)  
- Tratamiento de valores nulos según tipo de variable  
- Análisis exploratorio de datos (EDA)  
- Visualización con matplotlib y seaborn  
- Construcción de dashboard interactivo en Power BI  

## 📊 Resultados  
Se identificó que los salarios tienden a aumentar con el nivel de experiencia.  
Además, se observaron diferencias salariales según el rol y el país, así como una mayor concentración de empleos en determinadas regiones.

## 📈 Dashboard  
Se desarrolló un dashboard interactivo en Power BI que permite visualizar los principales indicadores del mercado laboral, incluyendo KPIs, distribución de salarios y filtros dinámicos.
